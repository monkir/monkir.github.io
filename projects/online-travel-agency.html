Online Travel Agency (OTA) System

Enterprise-level B2B & B2C Travel Platform

Project Overview

A comprehensive Online Travel Agency platform designed to manage real-time flight, hotel, and tour package bookings for both B2B agents and B2C customers. The system integrates external airline APIs and GDS services to ensure live fare, availability, and booking synchronization.

My Role & Responsibilities

  • Designed and developed high-performance ASP.NET Web APIs and MVC modules.
  • Integrated GDS and Airline APIs for real-time booking.
  • Implemented full booking lifecycle (Search → Book → Issue → Reissue → Cancel → Refund).
  • Developed role-based access for Admin, Agents, and Customers.
  • Optimized complex SQL stored procedures for high-speed transaction handling.

Key Features

  • Real-time flight search & booking
  • Hotel booking with dynamic pricing & availability
  • Holiday & tour package module
  • Secure payment & ticket issuance
  • Admin dashboard & reporting system

Technology Stack

ASP.NET MVC ASP.NET Web API Microsoft SQL Server Bootstrap jQuery

Technical Challenges & Solutions

Handling real-time booking synchronization required efficient API response handling and database transaction management. Optimized indexing strategies and stored procedures reduced query execution time significantly under high traffic.
